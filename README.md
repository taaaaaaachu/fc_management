# 固定費チェッカー（仮）

## サイト概要
### サイトテーマ
* 固定費を可視化する
* 他人の固定費の内訳や節約方法の紹介などをする
* コメント機能をつける

### テーマを選んだ理由
* いつも給料日に思っているより少ない振込に驚くので固定費を可視化できたら便利だと思った

### ターゲットユーザ
* 固定費を減らしたいと思っている人

### 主な利用シーン
* 支出の見える化をしたい時
